Для публикации в интернете моей версии Three.js. Почемуто Three.js fork публикует только файлы из репозитория mrdoob/three.js, с которого я сделала fork.
Внимание!!! Название репозитория должнобыть написано маленькими бкувами!
Для публикации в интернете надо открыть веб страницу репозитория https://github.com/anhr/mythree.js
Нажать Settings. Откроется веб страница Settings. 
Найти GitHub Pages. В группе Source Нажать кнопку None (Select a source below to enable GitHub Pages for this repository.) и нажать Save.
В параграфе GitHub Pages появится строка: 
Your site is ready to be published at https://anhr.github.io/mythree.js/.
